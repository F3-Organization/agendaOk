# Stitch Premium Design Prompt (ConfirmaZap)

Utilize este prompt no **Stitch** para gerar uma UI de alta fidelidade que herda toda a inteligência do nosso backend e as diretrizes de arquitetura sênior estabelecidas.

## 🛠 Contexto Técnico
- **Stack**: React (Vite) + Tailwind CSS + shadcn/ui.
- **Domínio**: `confirmazap.com`
- **Estética**: Minimalismo Premium (estilo Apple/Stripe), Dark Mode nativo, Glassmorphism, Micro-animações suaves.

## 📝 Prompt para o Stitch

> "Aja como um Senior Product Designer. Crie uma interface SaaS para o 'ConfirmaZap', uma plataforma de automação de confirmações de consultas via WhatsApp com sincronização total ao Google Calendar.
>
> **Diretrizes de Layout**:
> 1. **Sidebar**: Navegação minimalista (Dashboard, Agendas, WhatsApp, Assinatura).
> 2. **Header**: Badge dinâmico de status (Google: Conectado/Desconectado) e indicador de conexão Evolution API (WhatsApp).
> 3. **Dashboard Principal**:
>    - **Estatísticas**: Cards resumidos (Mensagens enviadas, Consultas confirmadas hoje, % de economia de tempo).
>    - **Tabela de Agendamentos**: Lista rica com badges de status coloridos. Filtros por período e busca por nome de cliente/telefone.
> 4. **Página de Integração WhatsApp**: Central de pairing com placeholder para QR Code, botão 'Desconectar' com confirmação visual e dicas de troubleshooting.
> 5. **Upgrade PRO**: Banner sutil de upsell para contas FREE, destacando os benefícios (Sincronização ilimitada, suporte prioritário).
>
> **Estilo Visual**: Use escalas de cinza ricas, sombras de baixa opacidade, fontes geométricas (Inter/Outfit) e cores de destaque Emerald (sucesso) e Purple (marca). O design deve ser funcional, limpo e transmitir confiança e automação profissional."
